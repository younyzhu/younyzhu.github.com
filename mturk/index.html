This study is a supplement of PathBubble for manually adjusting biological pathways. If you want to know detailed information, please visit our PathBubble website: PathBubble
Procedures: In this task, you will be shown a pathway graph.
1. You should carefully move the nodes(elements) of the graph, which should obey the following Rules.
2. During the operation, you can click the Left Crossing button on the menu control( see image below), It will tell you the number of edge crossing right now.
3. After finishing arranging the graph, you should find your amazon mechnical turk worker Id , and input your worker Id in the menu. And then press Send, it will come out an alert with you work Id, you should make sure you worker Id is correct, and then press Send again.
(If you can not find your worker Id, please visit https://www.mturk.com/mturk/dashboard, in you account dashboard, you can find you Worker Id)

Rules

  1. Avoid Edge crossing.
  2. Try to reduce the length of the edges.
  3. When moving the compartment, nodes inside it should not be moved.
  4. If the compartment has no connections with other compartment, it should be put on the top.
  5. In the final graph, the direction of the majority of the arrows should go down.
  6. All the nodes in a graph should not be overlaped.
  7. You should balance the trade between the beauty of the graph and uniform distribution of the nodes.
