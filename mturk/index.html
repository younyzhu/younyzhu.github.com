Description

This is a supplement tool of PathBubble for manually adjusting biological pathways. If you want to know detail information, please visit our PathBubble website: PathBubble In these task, you will be show a pathway graph. You should carefully move the elements of the graph which obey the following rules.

Rules

  1. First, Avoid Edge crossing
  2. Second, Try to reduce the length of the edges
  3. Third, When moving the compartment, the block nodes inside should not be moved
  4. Four, If the compartment has no connections with other compartment, it should be put on the top.
  5. Five, In the final graph, the direction of the majority of the arrows should go down.
  6. Six, the nodes in a graph should not be overlaped
  7. Seven, You should balance the trade between the beauty of the graph and uniform distribution of the nodes.
